-- Desafio 01 — CRUD com MySQL

Projeto desenvolvido como parte da **Formação Backend JS**

-- Sobre o projeto --

Prática completa de banco de dados relacional com MySQL, cobrindo criação de tabelas, manipulação de dados e consultas com filtros.

-- Conteúdo praticado --

- `CREATE TABLE` — criação de tabelas com tipos de dados
- `INSERT INTO` — inserção de registros
- `UPDATE` — atualização de dados
- `DELETE FROM` — exclusão de registros
- `ALTER TABLE` — modificação de estrutura
- `SELECT` com `WHERE`, `OR`, `LIKE` — consultas com filtros

-- Tabelas criadas --

- **usuarios** — nome, usuário, senha, idade
- **vendas** — usuário, produto, preço, quantidade, categoria

-- Tecnologias --

- MySQL
- MySQL Workbench
